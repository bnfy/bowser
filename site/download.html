before you install

What you're actually downloading.

macOS, signed and notarized

Every macOS build is code-signed and notarized with Apple, so Gatekeeper verifies it before first launch. This applies to the macOS builds specifically.

Updates arrive in place

Installed release builds check GitHub Releases and update themselves automatically — no separate updater, no account.

One launch ping, with an off switch

At launch, Blanc sends one ping: a random install ID, a random session ID, the app version, platform, and architecture. It carries no browsing history, URLs, searches, or page content. Turn it off in Settings → Privacy & Security under “Help improve Blanc,” or read the privacy policy.

a note for windows

Why might Windows warn about Blanc?

The current Windows installer is not yet code-signed, so Microsoft Defender SmartScreen may show an “unknown publisher” or “Windows protected your PC” notice while Windows signing is completed. Take that warning seriously wherever you see it: download Blanc only from this page or the official GitHub release, and skip copies hosted anywhere else.
