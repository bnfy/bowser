home/features/command palette

keyboard first

One shortcut to move through your whole session.

Press Command or Control L to jump to a tab, favorite, history item, or group. Type / when you want to run a browser command instead.

jump, don't hunt

The same input gets you to the next thing.

Find a tab with a few letters.

The Quick Switcher matches across open tabs, favorites, browsing history, and named groups. Choose a result to focus it without sorting through a tab strip.

Run a browser action with /.

Type / to see commands for everyday browser work, including /private, /find, /group work, and /allow-ads.

Keep the page where it is.

The command palette floats over the page instead of moving content to make room. Dismiss it with Escape when you are done.

one place for the little chores

Switch the session. Change the browser. Stay in context.

The command palette brings search, tab switching, and browser actions into one repeatable gesture. It is designed to shorten the trip between what you are reading and what you need to do next.

Bring up the controls only when you need them.
