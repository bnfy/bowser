# Blanc 1.0.0-rc.1

This is Blanc 1.0's signed Apple Silicon release candidate for active testing
and review. It is a prerelease and does not replace the current Stable update
for existing users.

## What to try

- **Optional vertical tabs.** Choose **View → Tab Layout → Vertical Tabs** or
  Settings → General → Tab layout. The Island remains the only address,
  search, and command surface; switching layouts does not reload open pages.
- **The Island.** Press Command-L to search, switch among tabs, find Favorites
  and history, reach tabs shared by another device, or run slash commands.
- **Named groups and private tabs.** Groups stay name-based rather than
  color-coded. Private tabs use a separate, non-persistent in-memory session
  and stay out of Blanc history, session restore, and reopen-closed.
- **Built-in blocking.** EasyList and EasyPrivacy filtering runs at the browser
  session layer rather than through an extension.

## Release-candidate hardening

- Blanc now presents usable local chrome while blocking filters initialize.
  A failed filter startup offers **Retry** and an explicit
  **Continue without blocking** path instead of hanging or silently weakening
  protection.
- On a fresh profile, Search suggestions and Help improve Blanc are both
  presented before either path may send. Both start enabled; either can be
  turned off before continuing or later in Settings.
- Chrome and internal-page fonts are bundled locally.
- The release pipeline stages and verifies the exact notarized artifact set,
  updater metadata, blockmaps, and SHA-256 checksums before publication.

## Availability and boundaries

- This candidate is for **macOS on Apple Silicon only**.
- Blanc has no extension runtime or password import.
- Ad and tracker blocking is best effort.
- Profile Sync is optional and end-to-end encrypted; open-tab sharing is a
  separate, off-by-default per-device snapshot.

Download `Blanc-1.0.0-rc.1-arm64.dmg` and verify it against `SHA256SUMS` below.
